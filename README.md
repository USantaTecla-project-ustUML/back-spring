## :gear: Ejecución en local
1. Ejecutar el motor de Postgresql: `//localhost:5432/tpv`
1. Ejecutar en consola: `mvn clean spring-boot:run`

* Cliente Web (OpenAPI): `http://localhost:8081/swagger-ui.html`
